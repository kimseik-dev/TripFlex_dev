# TripFlex Design System

## 1. Design Concept
- Mobile-first translation app for travelers.
- Clean, dark-mode UI with high-contrast elements.
- Precision coordinate mapping for translation overlays.

## 2. Layout
- **Fixed Size**: 390x844px (Standard mobile aspect ratio).
- **Style**: Floating app container in the center of the web viewport.

## 3. Colors
- **Background**: `#121212` / `#000000`
- **Accent**: `#FFE600` (Electric Yellow)
- **Overlay**: `rgba(255, 235, 59, 0.9)` (Semi-transparent yellow)

## 4. Components
- **Viewfinder**: Main area showing the scanned image.
- **Overlays**: Positional bubbles with translated text.
- **Bottom Nav**: History, Scan (FAB), Settings.
