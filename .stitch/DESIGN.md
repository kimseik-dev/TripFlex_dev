# TripFlex NexGen Design System 🎨✨

## 1. Brand Identity
TripFlex NexGen is a premium, high-tech travel companion. It uses futuristic elements like glassmorphism and glowing gradients to convey a sense of 'Digital Magic' and 'Reliable Intelligence'.

## 2. Color Palette 🌈
- **Primary**: Deep Indigo (`#1A1A2E`) & Azure Blue (`#16213E`)
- **Accent**: Neon Cyan (`#00D2FF`) & Soft Purple (`#9D50BB`)
- **Background**: Almost Black (`#0F111A`)
- **Glassmorphism**: White/Light Cyan with 10-20% opacity and 10px blur.

## 3. Typography ✍️
- **Primary Font**: `Inter` - Clean, modern, and highly legible.
- **Headings**: `Outfit` - Geometric and futuristic.
- **Weights**: Regular (400) for body, Bold (700) for headings.

## 4. Components & Patterns 🛠️
- **Buttons**: Rounded (Full), glowing borders, gradient fills.
- **Cards**: Glassmorphism with subtle borders, soft shadows.
- **Overlays**: Semi-transparent floating panels with blur.
- **Icons**: Outlined, thin stroke, neon cyan color.

## 5. Spacing & Grid 📏
- **Mobile Grid**: 20px side margins, 12px or 16px vertical gutters.
- **Spacing Units**: 4px base (4, 8, 12, 16, 24, 32, 48, 64).

## 6. Design System Notes for Stitch Generation 🪄
When generating next screens for TripFlex, use the following prompt block:
> **DESIGN SYSTEM (REQUIRED):**
> Futuristic mobile dark mode. Palette: Indigo, Azure, Neon Cyan, Purple.
> Style: Glassmorphism, 10px blur, glowing accents, smooth gradients.
> Font: Modern Sans-serif (Inter/Outfit).
> Layout: Mobile-first (390x844), premium feel, minimal but impactful.
