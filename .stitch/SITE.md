# TripFlex NexGen Site Vision & Roadmap 🚀✨

## 1. Vision
TripFlex NexGen is more than just a translator; it's a premium, AI-powered travel companion. Leveraging Azure AI Vision and Translator, it provides real-time OCR and instant Korean translations with a futuristic, glassmorphism UI.

## 2. Tech Stack
- **Frontend**: Next.js (App Router), React, Vanilla CSS
- **Backend**: NestJS, Node.js
- **Design**: Stitch MCP (Futuristic Mobile-First)
- **AI**: Azure AI Vision (OCR), Azure AI Translator
