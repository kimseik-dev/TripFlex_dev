# Azure AI Text Translation Skill

This skill provides access to the Azure AI Text Translation SDK, which allows for real-time translation and language analysis.

## Key Features
- **Translation**: Translate text into one or more languages.
- **Transliteration**: Convert text from one script to another.
- **Language Detection**: Detect the language of a given text.
- **Dictionary Lookup**: Get dictionary definitions and translations.

## Usage
To use this skill, ensure you have an Azure AI Translator resource and the following environment variables:
- `VITE_AZURE_TRANSLATOR_ENDPOINT`
- `VITE_AZURE_TRANSLATOR_KEY`
- `VITE_AZURE_TRANSLATOR_REGION`
