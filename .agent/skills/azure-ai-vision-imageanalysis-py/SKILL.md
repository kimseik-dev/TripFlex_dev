# Azure AI Vision Image Analysis Skill

This skill provides access to the Azure AI Vision Image Analysis SDK, which allows for advanced image processing and analysis.

## Key Features
- **OCR (Optical Character Recognition)**: Extract text from images.
- **Captions**: Generate descriptive captions for images.
- **Tags**: Identify objects and themes in images.
- **Objects**: Detect and locate common objects.
- **People Detection**: Identify people in images.
- **Smart Cropping**: Generate thumbnails and crop regions.

## Usage
To use this skill, ensure you have an Azure AI Vision resource and the following environment variables:
- `VITE_AZURE_VISION_ENDPOINT`
- `VITE_AZURE_VISION_KEY`
